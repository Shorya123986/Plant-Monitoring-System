Plant Monitoring System (IoT)
An IoT solution to monitor soil moisture, temperature, humidity, and light levels for plants. It automatically activates irrigation based on soil moisture data.

Features
Real-time environmental monitoring
Automatic water pump control
Data visualization via cloud dashboard

Hardware
NodeMCU/Arduino
Soil Moisture Sensor
DHT11/DHT22 Sensor
Light Sensor (LDR)
Relay and Water Pump

Setup
Connect sensors to microcontroller pins
Upload code via Arduino IDE
Configure Wi-Fi and cloud platform
Monitor data and control irrigation remotely

Usage
Automatically waters plants when soil moisture is low, improving plant care and waterÂ efficiency.
